2.4. Бағдарламалау

Математикалық алгоритмдерді құру жолындағы бірінші қадам бағдарлама құрылымын және математикалық операциялар жиынтығын, бағдарламалау тілін үйрену болып табылады. Сонымен қатар, осы тарауда математикалық операциялар мен MatLab жүйесінің скалярды және матрицалы айнымалыларды өңдеуге байланысты функциялары қарастырылады. Әдетте, MatLab-тағы әрбір бағдарлама функцияны білдіреді және бос орын арқылы оның атауы берілетін function кілт сөзінен басталады. Мысалы: 

                function Lab1 
                a = 5; 
                b = 2; 
                c = a * b;

Бұл бағдарлама Lab1 деп аталатын функцияға негізделген және екі А және b айнымалыларының көбейтіндісін есептейді. M-файлда бағдарламаны сақтау кезінде функцияның атауымен сәйкес келетін файл атауын көрсету ұсынылады, яғни бұл жағдайда – Lab1. Бір m-файлда көптеген қосымша мүмкіндіктер сұралатынын атап өту керек. Ол үшін негізгі бағдарламаның листинг соңында тағы бір кілт сөзін жазып, оның атын, қою жеткілікті,мысалы: 

                function Lab1 
                a = 5; 
                b = 2; 
                c = a * b; 
                out_c (c); % out_c функциясын шақыру() 
                function out_c ( arg_c) % out_c функциясын анықтау() 
                disp(arg_c);

Out_c () функциясын негізгі бағдарламада оны анықтағанға дейін шақыруға болатынына назар аударыңыз. Бұл функцияны тапсырманың реттілігі туралы бағдарламашыны алаңдатпауға мүмкіндік беретін MatLab тілінің ерекшелігі. Келтірілген мысалда out_c() функциясы енгізілген disp () функциясы арқылы экранға (MatLab командалық терезесіне) шығатын arg_c атымен бір кіріс параметріне ие. Нәтижесінде, келтірілген бағдарламаны орындау кезінде MatLab командалық терезесінде c айнымалысының мәні көрсетіледі.

1-ші файл (Lab1.m)
function Lab1
a = 5;
b = 2;
c = square(a,b); % вызовфункции square()
out_c(c); % вызовфункцииout_c()

function out_c(arg_c) % определениефункцииout_c()
disp(arg_c);
  \
-ші файл (square.m)
function res=square(a, b)
res = a*b;

Lab1 функциясын орындау кезінде MatLab жүйесі square файлынан square функциясын шақырады.m. Бұл автоматты түрде жасалады, себебі MatLab тілінің кірістірілген функциялары, сондай-ақ атаулары, әдетте, шақырылатын функциялардың аттарына сәйкес келетін файлдардан анықталған және шақырылады. Сондай-ақ, square() функциясы a және b екі аргументін қабылдайды ғана емес, сонымен қатар RES айнымалысы арқылы олардың жұмысын қайтарады. Ұсынылған синтаксис негізгі бағдарламаға есептеу нәтижесін қайтару қажет болған кезде қолданылуы керек. Бұл құралдың төртінші тарауында әртүрлі алгоритмдерді іске асыру үшін функцияларды шақырудың конструкциялары егжей-тегжейлі жазылған.
MatLab-та қарапайым айнымалылар және негізгі деректер түрлері

Бағдарламаны құру, әдетте, айнымалыларды анықтаудан және деректерді ұсыну тәсілінен басталады. Сондықтан, осы бағдарламаның сипаттамасын дұрыс ұйымдастыру үшін MatLab-та айнымалыларды қалай қою керектігін және қандай айнымалылардың түрлері мүмкін екенін білу қажет. Ең қарапайым және ең көп таралған деректер түрі - бұл сан. MatLab - да сан бірнеше бірегей аты бар айнымалы сақталады, мысалы,
a = 5;
a атымен айнымалыны орнатады және оған 5 мәнін береді. Әдепкі бойынша a айнымалысы нақты (double түрі), яғни, мысалы, бөлшек мәндерді қабылдай алады,
a = -7.8;
a тең айнымалы мәнін көрсетеді -7,8. Айнымалы түрін тиісті кілтсөз арқылы берілетін сан түрін көрсету арқылы өзгертуге болады, мысалы,
a = int16(5);
5 санын бүтін 16 биттік мән ретінде белгілеуді орындау. Мұндай операцияны орындау нәтижесінде a айнымалы түрі int16 сәйкес болады.
MatLab-та қол жетімді деректер түрлері кестеде көрсетілген.
. 2.1.

Кесте 2.1. MatLab негізгі деректер түрлері

Әдепкі бойынша, зат санының ең көп дәлдігі бар double түрі қолданылады және әмбебап түрі болып табылады. Алайда, егер ЭЕМ жадын үнемдеу қажет болса, онда өзіңіз қалаған түрін көрсетуге болады [8]. Айнымалыларды беру кезінде соңғы нәрсе-олардың аттарын анықтау ережесі. MatLab-да айнымалылардың аттары тек латын әріптерімен, сандармен және " _ " символымен берілуі мүмкін. Ал бірінші таңба латын әліпбиінің әрпіне сәйкес келуі керек. Сондай-ақ, атап өткен жөн, бұл аттар 

                arg = 1; 
                Arg = 2; 
                ARG = 3;

бұл үш түрлі атау, яғни тиісінше 1, 2 және 3 мәндері бар үш түрлі айнымалылар. Бұл мысал MatLab регистр айнымалылардың атауларында ажыратады. Бағдарламалау кезінде қандай мәліметтерді түсінуге болатын айнымалылардың мағыналы аттарын қою жақсы. Бұл үлкен бағдарламаларды құру кезінде шатасуды болдырмауға мүмкіндік береді.

Қарапайым айнымалылармен арифметикалық операциялар MatLab-да қарапайым айнымалылардың негізгі арифметикалық операцияларын қарастырайық. A және b екі айнымалысы болсын. Сонда қосу, азайту, көбейту және бөлу операциялары осылай жазылады: 

                c = a+b; % қосу 
                c = a-b; % азайту 
                c = a * b; % көбейту 
                c = a / b; % бөлу

Сонымен қатар, MatLab-да осылай жазылған дәрежеге тұрғызу операциясы анықталды:
c = a^2; % a айнымалысын квадратқа тұрғызу
c = a^0.5; % A айнымалысынан квадрат түбірін шығару

Арифметикалық операциялар * және / жоғары + және -, яғни алдымен көбейту және бөлу, содан кейін қосу және азайту орындалады. Құру операциясы ең жоғары басымдыққа ие. Арифметикалық операциялардың басымдықтарын өзгерту үшін қарапайым математикадағы сияқты дөңгелек жақшалар қолданылады, мысалы, 

                c = 7+2*2; % C = 28 мәні 
                c = (7+2)*2; % с = 18 мәні

Бағдарламалау тәжірибесінде арифметикалық операцияларды пайдаланудың бірнеше тұрақты тәсілдері бар. Мысалы, arg айнымалы мәнін 1-ге арттыру қажет болса, оған келесідей қол жеткізуге болады:
arg = arg + 1;
Бұл кезде arg айнымалысына тең нәрсе маңызды емес, кез келген жағдайда оның мәні 1-ге ұлғайтылады. Сол сияқты кез келген шамаға айнымалы мәнінің ұлғаюын немесе азаюын орындауға болады.
Бағдарламалаудың басқа тұрақты тәсілі екі айнымалы арасындағы мәндерді алмасу болып табылады. Мысалы, arg_a және arg_b айнымалылары берілген және олардың арасында деректер алмасуды жүргізу қажет. Бұл келесідей қол жеткізіледі: 

                temp = arg_a;
                arg_a = arg_b;
                arg_b = temp;

Мұнда temp-arg_a айнымалы мәнін сақтау үшін қажетті уақытша айнымалы, өйткені ол екінші жолда arg_b мәнімен сүртіледі. Сондықтан arg_b айнымалы үшінші жолда temp сақталған arg_a айнымалы мәні беріледі. Егер есептеулерді орындау нәтижесінде комплекс сан пайда болса, онда MatLab комплекс Сандардың арифметикасына сәйкес осындай сандармен автоматты түрде әрекет етеді. Мысалы, 1-ден квадрат түбірін алу кезінде келесі нәтиже аламыз: 

             
                c = (-1)^0.5; % c = 0.0000 + 1.0000

Мұнда i-сақталған жалған бірліктің атауы және ұсынылған жазба нөлдік нақты бөлігі бар және бірлі-жарым жалған кешенді санды білдіреді. MatLab-да жалған бірлік ретінде сақталған j әрпі қолданылады. Кешенді айнымалыны орнату үшін олардың нақты және жалған бөліктерінің мәнін төменде көрсетілгендей көрсету жеткілікті

C = 6 + 5i; % кешенді сан Кешенді сандармен жұмыс істеу кезінде екі арнайы функция бар:
real (x) – кешенді санның нақты бөлігін алу x;
image (x) – кешенді санның жалған бөлігін алу x;
abs (x) – кешенді санның абсолюттік мәнін есептеу x;
conj (x) – кешенді-жұптасқан x санын есептеу;
angle (x) - x кешенді санның аргументін есептеу;
MatLab негізгі математикалық функциялар
MatLab алгоритмдерді жүзеге асыру кезінде олардың аты бойынша қол жетімді барлық кең таралған математикалық функцияларды қамтиды. Мысалы, sqrt() функциясы сан квадратын есептеп береді және бағдарламада келесі түрде қолданылуы мүмкін: 

                x = 2; 
                y = 4; 
                d = sqrt (x^2+y^2); % Евклид қашықтығын есептеу

Кесте бойынша берілген барлық басқа математикалық функциялар да
ұқсас. Кесте 2.2. MatLab негізгі математикалық функциялары

Барлық қарапайым функциялар кешенді дәлелдермен де есептеулерге жол береді. Мысалы:
res = sin(2+3 i) * atan(4i)/(1 — 6i); % rus = -1.8009 — 1.9190 i
немесе
exp (i*x) = cos (x)+i * sin(x);
Төменде a аты бар Вектор тапсырмасының мысалы көрсетілген және 1, 2, 3, 4 мәндері бар:
a = [1 2 3 4]; % вектор-жол
Вектордың қандай да бір элементіне кіру үшін тілдің келесі құрылымы қолданылады:
disp(a (1) ); % вектордың 1 элементінің мәнін көрсету
disp(a (2) ); % вектордың 2 элементінің мәнін көрсету
disp(a (3) ); % вектордың 3 элементінің мәнін көрсету
disp(a (4) ); % вектордың 4 элементінің мәнін көрсету
яғни, вектордың атауын көрсету керек және дөңгелек жақшаларда жұмыс істеуге болатын элементтің индекс нөмірін жазу керек. Мысалы, 2-ші массив элементінің мәнін 10-ға өзгерту үшін жазу жеткілікті
a (2) = 10; % 2 элементінің мәнін 10-ға өзгерту
Вектордағы элементтердің жалпы санын анықтау қажеттілігі жиі туындайды, яғни оның мөлшерін анықтау. Бұл length() функциясын пайдаланып жасауға болады:
N = length(a); % (n=4) А массив элементтерінің саны
Егер вектор-бағанды орнату қажет болса, оны осылай жасауға болады
a = [1; 2; 3; 4]; % вектор-баған
немесе
b = [1 2 3 4]'; % вектор-баған
бұл ретте векторлар элементтеріне қолжетімділік векторлар-жолдар үшін де жүзеге асырылады.
Векторлар тек жекелеген сандардан немесе айнымалылардан ғана емес, сонымен қатар векторлардан да құрайтынын атап өткен жөн. Мысалы, келесі бағдарлама фрагменті бір векторды екіншісінің негізінде қалай жасауға болатынын көрсетеді:
a = [1 2 3 4]; % бастапқы вектор a = [1 2 3 4]
b = [a 5 6]; % екінші вектор b = [1 2 3 4 5 6]
Мұнда B векторы алты элементтен тұрады және А векторы негізінде құрылған. Бұл әдісті пайдалана отырып, бағдарлама жұмысы барысында векторлардың мөлшерін арттыруға болады:
a = [a 5]; % А векторының бір элементке ұлғаюы
Векторларды тапсырманың (инициализациялаудың) сипатталған тәсілінің кемшілігі, мысалы, 100 немесе 1000 элементтен тұратын үлкен өлшемді векторларды анықтаудың қиындығы болып табылады. Бұл тапсырманы шешу үшін MatLab векторларын нөлдермен, бірліктермен немесе кездейсоқ мәндермен инициализациялау функциялары бар:
a1 = zeros (1, 100); % вектор-жол, 100 элемент
% нөлдік мәндер
a2 = zeros (100, 1); % вектор-баған, 100 элемент
% нөлдік мәндер
a3 = ones (1, 1000); % вектор-жол, 1000 элемент
бірлі-жарым мәндер %
a4 = ones (1000, 1); % вектор-баған, 1000 элементтері
бірлі-жарым мәндер %
a5 = rand (1000, 1); % вектор-баған, 1000 со элементтері
% кездейсоқ мәндер.
Матрица амалдары және векторларды қолдану
MatLab жүйесінде матрицалар мен векторларға математикалық операциялар орындалады. Бірінші матрицалар мен векторларды қосу және көбейтудің қарапайым операцияларын қарастырайық. Екі вектор болсын
a = [1 2 3 4 5]; % вектор-жол
b = [1; 1; 1; 1; 1]; % вектор-баған
сонда осы екі векторды көбейту осылай жазуға болады
c = a * b; % c=1+2+3+4+5=16
D = B*a; % d – 5х5 элементтің матрицасы
Векторлардың үстінен операцияларға сәйкес, вектор-жолды вектор-бағанға көбейту сан береді, ал вектор-бағанды вектор-жолға көбейту екі өлшемді матрицаны береді, бұл келтірілген мысалдағы есептеулердің нәтижесі болып табылады, яғни
Екі векторды қосу және азайту осылай жазылады
a1 = [1 2 3 4 5];
a2 = [5 4 3 2 1];
c = a1+a2; % c = [1+5, 2+4, 3+3, 4+2, 5+1];
с = a2-a1; % c = [5-1, 4-2, 3-3, 2-4, 1-5];
Қосу және азайту операцияларын екі вектор-бағандар немесе екі вектор-жолдар арасында орындауға болатынына назар аудару керек. Әйтпесе MatLab қате туралы хабарлама береді, өйткені әртүрлі векторларды бүктеуге болмайды. Барлық жол берілмейтін арифметикалық операцияларға қатысты жағдай: оларды есептеу мүмкін болмаған жағдайда MatLab жүйесі қате туралы хабарлайды және бағдарламаны орындау тиісті жолда аяқталады.
